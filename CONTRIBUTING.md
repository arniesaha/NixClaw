# Contributing to NixClaw

Thanks for your interest in contributing! Here's how to get started.

## Pull Requests

1. Fork the repo and create your branch from `main`
2. Make your changes
3. Test on a real device (simulator doesn't support all features)
4. Submit a PR with a clear description of what changed and why

## Issues

Use GitHub Issues to report bugs or request features. Include:
- What you expected to happen
- What actually happened
- Device, iOS version, and Xcode version
- Steps to reproduce

## Development Setup

See the [README](README.md) for build instructions.

## What Goes Where

- **OpenClaw-specific features** → This repo (NixClaw)
- **Background mode, audio fixes, general improvements** → Consider also contributing upstream to [VisionClaw](https://github.com/sseanliu/VisionClaw) (requires Meta CLA)
- **DAT SDK issues** → [Meta Wearables DAT SDK](https://github.com/facebook/meta-wearables-dat-ios/discussions)
